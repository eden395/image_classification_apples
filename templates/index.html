Apple Freshness Classification System

A simple and formal web application that classifies uploaded apple images as either fresh or rotten using your trained Random Forest model and the same handcrafted feature extraction pipeline used in your notebook.

Project Summary

  • Model: Random Forest
  • Classes: Fresh and Rotten
  • Feature Types: RGB, HSV, LAB, GLCM, edge density
  • Deployment Target: Render

About the Project

This system applies traditional machine learning for apple image classification. It extracts color, texture, and edge-based features from an uploaded image, applies the saved preprocessing pipeline, and returns the model prediction through a Flask backend.

Objective

To automatically determine whether an apple image belongs to the fresh or rotten category.

Model Basis

The deployed backend follows the same imputer, scaler, selector, label encoder, and model flow used during training.

Use Case

Suitable for laboratory demonstrations, academic presentations, and a simple machine learning deployment portfolio.
